# Results

Benchmark outputs and charts generated by the Agentic AI Stress Test Suite.

## Structure

```
results/
├── charts/
│   ├── article_01/    # Recall@5 bar, latency box, accuracy vs cost scatter
│   ├── article_02/    # Accuracy vs latency scatter, reranker comparison table
│   ├── article_04/    # Success rate, tool histogram, latency distribution
│   ├── article_05/    # Completion rate, token efficiency, latency
│   ├── article_06/    # Cumulative cost, cost breakdown, cache savings
│   ├── article_08/    # Throughput, p95 latency, parallel speedup
│   └── article_09/    # Recall comparison, UMAP, PyTorch speedup, JAX vs PyTorch
├── data/
│   ├── article_01_benchmarks.json
│   ├── article_02_benchmarks.json
│   ├── article_04_benchmarks.json
│   ├── article_06_benchmarks.json
│   ├── article_07_benchmarks.json
│   ├── article_08_benchmarks.json
│   ├── article_09_benchmarks.json
│   └── pytorch_optimizations.json
└── reports/
    └── v1.0_summary.md   # Aggregated findings across all 9 articles
```

## Regenerating

```bash
# Regenerate all charts (executes Jupyter notebooks)
./scripts/generate_all_charts.sh

# Individual article benchmark
uv run python benchmarks/run_article_04.py --mock
```
